
## Architecture Overview

```
Level4View (Main Container)
├── DualRingMechanism (SVG Rings)
│   ├── Outer Ring: SubNodes (Input, Output, Processing, Model...)
│   └── Inner Ring: MiniNodes (Device, Sensitivity, Noise Gate...)
├── Center Button: Back to categories
├── DetailPanel (Side Panel)
│   ├── Glowing connection line
│   ├── Form fields (memoized components)
│   │   ├── ToggleField
│   │   ├── SliderField
│   │   ├── DropdownField (with loadOptions)
│   │   ├── TextField
│   │   └── ColorField
│   └── Confirm button
└── Animation layers (glow, shimmer, flash)
```

### Navigation State
- `selectedSubNodeIndex`: Controls outer ring rotation
- `selectedMiniNodeIndex`: Controls inner ring rotation
- `confirmFlash/confirmSpinning/lineRetracted`: Animation states

## Performance Optimizations

1. **React.memo** on all field components and main mechanisms
2. **useMemo** for expensive calculations (arc paths, segment angles)
3. **useCallback** for event handlers to prevent re-renders
4. **Dynamic imports** for icons (tree-shakeable)
5. **Lazy option loading** via `loadOptions` for WebSocket-backed dropdowns

## Usage Example

```tsx
import { Level4View, NAVIGATION_DATA } from "./Level4View"
import { useBrandColor } from "@/contexts/BrandColorContext"

function SettingsPage() {
  const { getThemeConfig } = useBrandColor()
  const theme = getThemeConfig()
  const [activeCategory, setActiveCategory] = useState("voice")
  const [view, setView] = useState<"categories" | "level4">("categories")

  if (view === "level4") {
    return (
      <Level4View
        categoryId={activeCategory}
        glowColor={theme.glow.color}
        expandedIrisSize={400}
        initialValues={{
          // Pre-populate from backend
          agent_name: "Iris",
          tts_speed: 1.2
        }}
        onConfirm={(values) => {
          // Persist to backend
          await saveSettings(values)
          setView("categories")
        }}
        onBackToCategories={() => setView("categories")}
      />
    )
  }

  return <CategorySelection onSelect={(id) => {
    setActiveCategory(id)
    setView("level4")
  }} />
}
```

## Dynamic Dropdowns (WebSocket)

For fields requiring backend data:

```typescript
{
  id: "input_device",
  label: "Microphone",
  type: "dropdown",
  loadOptions: async () => {
    // Connect to WebSocket/backend
    const devices = await fetchAudioDevices()
    return devices.map(d => d.name)
  }
}
```

The component automatically calls `loadOptions` when the mini-node mounts and caches results.

## Animation Sequence

**Selection Change:**
1. Spring rotation to center item at 12 o'clock (stiffness: 80, damping: 16)
2. Panel crossfade (opacity + y translation)
3. Line extension from orb to panel (spring)

**Confirm:**
1. Line retraction (spring)
2. Counter-spin outer ring (+360°) and inner ring (-360°)
3. Flash overlay scale pulse (0.8 → 1.4)
4. Glow breathe intensifies (scale 1 → 2.2)
5. After 900ms: onConfirm callback
